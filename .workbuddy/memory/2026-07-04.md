## Phase 0 职业与行业字段冗余分析

- 用户质疑 Phase 0 注册阶段 `occupation`（职业）和 `industry`（行业）两个问题重复
- 经查证，两个字段选项重叠率 80%（10个职业选项中8个在行业里有近 identical 对应项）
- 更严重的问题：选项字面还不一致（金融 vs 金融/投资），制造认知摩擦
- 最关键发现：`industry` 字段在后端核心函数（`buildTextForEmbedding`、`buildPersonaSeedFromSurvey`）中完全未被引用，是"死数据"
- Phase 0 完成校验只检查 `id.occupation`，未校验 `id.industry`
- 建议合并为单一字段，14 个去重选项，对下游零影响
- 分析报告：`outputs/Phase0-Occupation-Industry-Redundancy-Analysis.md`
- 关键文件：`Echo/src/features/onboarding/v2/phase0-fields.data.ts`（第 130-158 行定义了两个字段）
