## Phase 2 对话 Agent 重新设计（老许→阿辰 + 全异性化）

### 背景
用户指出小鹿(死党)和老许(深交老友)关系类型重合，都是"平等朋友"象限。Echo 是陌生人交友软件，不需要家人/长辈/上下级关系。最终方案：保留阿远/小鹿/小夜，将老许替换为阿辰（有过好感但让你失望的人），采集失望/冲突语气。

## UI 配色方案更换：紫色→蓝色（#2B8AEF）

### 背景
用户认为白+紫的配色像 AI 生成的产品，与吉祥物（蓝色+粉色）风格不匹配。保留白色背景，将按钮/交互色从紫色更换为蓝+粉。

### 颜色映射
- `#630ed4` (主UI紫) → `#2B8AEF` (天空蓝)
- `#7c3aed` (Auth紫/violet-600) → `#2B8AEF` (统一蓝色)
- `rgba(99,14,212,X)` → `rgba(43,138,239,X)` (半透明蓝色)
- `rgba(99, 14, 212, X)` → `rgba(43, 138, 239, X)` (boxShadow空间格版本)
- `rgba(124,58,237,X)` → `rgba(43,138,239,X)` (violet半透明)
- `#eef4ff` (浅蓝表面) → `#E8F4FF` (更蓝的浅蓝)
