## 帖子预览可点击 + 滚动位置恢复

用户确认要补上帖子预览可点击功能，并要求返回时保持滚动位置。

### 改动文件

1. **修改** `echo/src/features/profile/UserProfileView.tsx`:
   - +`useRef` import + `onOpenPost` prop
   - +`scrollRef` + `scrollKey = user-profile-scroll-${userId}`
   - +恢复滚动 useEffect（loading 完成后 rAF 恢复 scrollTop）
   - 滚动容器 `<div>` 加 `ref={scrollRef}`
   - 帖子预览 `<div>` → `<motion.button whileTap={{ scale: 0.98 }} onClick={保存scrollTop + onOpenPost}>`
   - +`line-clamp-3` 帖子内容截断（视觉提示可展开）
2. **修改** `echo/src/App.tsx` — UserProfileRoute 传 `onOpenPost={(postId) => navigate('/post/${postId}')}`

### 防误触设计

- 用 `whileTap={{ scale: 0.98 }}` 做纯视觉按下反馈
- **不用** `initial`/`animate` 入场 transform（避免 transform 干扰 touch 坐标）
- 浏览器原生 scroll vs click 判定（>10px 移动 = scroll）

### 滚动恢复机制

- 点击帖子时 `sessionStorage.setItem(scrollKey, scrollTop)`
- 返回时 useEffect 检测 `!loading` → rAF 恢复 `scrollRef.current.scrollTop`
- 按 `userId` 区分不同用户的滚动位置
